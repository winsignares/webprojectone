Sobre Mí

Soy estudiante de sexto semestre de Ingeniería de Sistemas en la Universidad Libre de Barranquilla, apasionada por la intersección entre la tecnología y las personas. Para lograrlo, utilizo las herramientas fundamentales del desarrollo web como HTML, CSS y JavaScript. Actualmente, estoy expandiendo mis habilidades de manera activa, aprendiendo Bootstrap para agilizar el diseño responsivo, así como Python y Java para adentrarme en el desarrollo de soluciones más complejas. Mi objetivo es unirme a un equipo donde pueda aplicar mis conocimientos y seguir este camino de aprendizaje para crear productos que la gente ame utilizar.
